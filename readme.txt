JFox for Google Chrome
Extension for Google Chrome that, with help of one Joomla Plugin, and one 
optional Joomla component, make more ease for Joomla programers and 
developers debug yours extensions, or even help with some possible security 
problems on your Joomla site.

----------------------------------- How to use ---------------------------------

1. Go to Joomla CMS site, with installed and enabled JFox plugin for Joomla and 
do login as a Super User
2. Press F12 button and go to Google Chrome console and navigate on site
    TIP: Choose "Preserve log upon navigation" for have one history


--------------------------- Instalation & Dependences --------------------------
Dependences:
    - Joomla CMS 1.7+:
        - JFox Plugin (for CMS Joomla) JFox
            - Installed
            - Enabled
    - Most Recent Google Chrome (tested on 15):
    - Go go "chrome://flags/" and enable use of experimental extensions (APIs de
        extensões experimentais, in portuguese)

Instalation from compiled source:
    - Just use .crx

Instalation from source:
    - Google Chrome > Tools > Extensions(?) > Load extension(?) > ***Choose 
        directory***
    - Google Chrome > Ferramentas > Extensões > Carregar extensão expandida > 
        ***Choose directory*** (in portuguese)
